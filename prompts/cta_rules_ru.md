CTA должен вытекать из смысла поста.

flight_deal: маршрут, цена, дата/период, условия, кнопка на конкретный поиск билетов или источник. Без философии.
destination_post/weekend_trip/city_break: мягко — билеты, отели, экскурсии, TourJin только если уместно.
tour_offer: туры и, если нужно, страховка или TourJin.
hotel_post: отели, билеты только если логично.
event_trip: событие, затем билеты и отели только если событие реально повод для поездки.
visa_or_residence/payment_abroad: карты/страховка/условия, без авиабилетов.
inspiration_story: можно без ссылок.

Не вставляй больше 3 кнопок.

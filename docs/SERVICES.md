# Сервисы и партнёрские ссылки

Источник истины: `configs/services.json`.

Правила:

- ссылки подставляются только по смыслу жанра;
- обещание кнопки должно совпадать с тем, куда ведёт URL;
- при travel-offer приоритет у собственных партнёрских ссылок;
- Telegram-кнопки на чужие каналы запрещены, кроме внутреннего `tourjin_bot`.

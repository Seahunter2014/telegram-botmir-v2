# v3.3 EDITORIAL_FORMAT_FIX

Исправления после проверки публикаций в Telegram:

1. Пост с картинкой теперь публикуется как фото с caption, а не как отдельная картинка и отдельный текст.
2. Caption ограничен до безопасной длины, чтобы Telegram не обрезал текст.
3. Заголовок форматируется жирным через HTML.
4. В пост добавляется умеренный жанровый эмодзи.
5. Усилен финальный редакционный слой: слабые варианты переписываются перед preview/publish.
6. Для слабых сигналов введён потолок score, чтобы бот не выбирал мусорные карточки сайтов.
7. CTA-кнопки переименованы в человеческие: билеты, отели, событие, туры.
8. Для событий первая кнопка ведёт на исходное событие/источник.
9. Для рейсов строится Aviasales deeplink, если распознаны IATA и дата.
10. Pexels ищет узнаваемые city landmark фото, orientation=landscape.

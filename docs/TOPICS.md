# Темы и жанры

Источник истины: `configs/topics.json`.

Ключевые рабочие жанры редакции:

- `destination_post`
- `weekend_trip`
- `flight_deal`
- `tour_offer`
- `hotel_post`
- `event_trip`
- `visa_or_residence`
- `payment_abroad`
- `practical_travel`

Дополнительные жанры допускаются только если не ломают слот, ротацию и логику CTA.

# CHANGELOG v5.5

Исправлена логика выбора тем для `/test N` и автопостинга.

- `source_manager.fetch_signals()` теперь возвращает сигналы round-robin по источникам, а не пачкой из первого источника.
- `/test 1`, `/test 2`, `/test 3` используют редакционный цикл жанров и не залипают на Vandrouki.
- `run_autopost()` сначала ранжирует все кандидаты с учётом слота, score и rotation, а не публикует первый подходящий сигнал.
- Telegram preview-photo не используется как готовое медиа.
